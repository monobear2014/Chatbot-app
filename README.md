# 📄 PDF RAG Chatbot

A simple Retrieval-Augmented Generation (RAG) chatbot built with Streamlit. Users can upload PDF documents, ask questions about the content, and receive context-aware answers generated by a Large Language Model (LLM).

## 🚀 Features

- Upload one or multiple PDF documents
- Extract and split text into chunks
- Store document embeddings in a vector database
- Retrieve the most relevant context for user queries
- Interactive chat interface built with Streamlit

## 🛠️ Tech Stack

- Python
- Streamlit
- ChromaDB
- PyPDF
- Ollama (LLM & Embedding)
- RAG (Retrieval-Augmented Generation)
